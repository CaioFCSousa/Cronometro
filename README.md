# Cronometro


Este projeto é um cronômetro simples desenvolvido com HTML, CSS e JavaScript. Ele permite ao usuário iniciar, pausar e resetar a contagem do tempo diretamente no navegador.

## Funcionalidades
- Iniciar o cronômetro
- Pausar o cronômetro
- Resetar o cronômetro

## Estrutura do Projeto
```
index.html         # Página principal do cronômetro
LICENSE            # Licença do projeto
README.md          # Documentação do projeto
css/styles.css     # Estilos visuais
js/scripts.js      # Lógica do cronômetro
```

## Como usar
1. Abra o arquivo `index.html` em seu navegador.
2. Utilize os botões para controlar o cronômetro.

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript
